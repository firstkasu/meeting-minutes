# 1. Ideate · idea.md

## 한 문장 아이디어
회의를 누르면 녹음 → 끝나면 자동으로 회의록까지 나오는 가장 단순한 도구.

## 핵심 동기
- 회의록 작성이 매번 반복되는 단순 노동인데, 정작 회의 본질에 시간을 못 씀
- 기존 솔루션(Otter, Clovanote 등)은 무겁고, 사내·로컬에서 돌리기 어렵고, 가격이 부담
- 마이크 + 스피커 출력(상대방 목소리)을 **동시에** 잡는 도구가 의외로 적음
- 회의가 끝나는 순간 **구조화된 마크다운 회의록**이 손에 들어오는 경험을 만들고 싶음

## 사용자 시나리오
> 화상회의(Zoom/Meet)를 시작하면서 앱의 [회의 시작] 버튼을 누른다.
> 회의가 끝나면 [회의 종료]를 누르고, 잠시 기다리면
> "# 대제목 / ## 논의 내용 / ## 액션 아이템 / ## 추후 계획" 형식의 회의록이 화면에 뜬다.
> 그대로 복사하거나 .md 파일로 다운로드해 슬랙·노션에 붙여 넣는다.

## 기능적 골자 (2개 버튼만)
1. **회의 시작**: 마이크 + 시스템 오디오 동시 녹음 시작
2. **회의 종료**: 녹음 중지 → 자동 전사 → LLM이 구조화된 회의록 생성

## 비기능적 신념
- **버튼은 2개만.** 옵션을 늘릴수록 사용 안 함
- **오프라인 동작 가능해야 함** (로컬 버전). 폐쇄망·기밀 회의 대응
- **언제든 LLM을 갈아끼울 수 있어야 함.** Anthropic ↔ Gemini ↔ OpenAI ↔ 로컬 LLM
- **데이터 보호**: API가 실패해도 전사 원본과 녹음 파일은 무조건 보존

## 출발점 가정
- 회의 최대 5분 미만 가정 (테스트용)
- Windows 11 환경 우선
- 한국어 회의 가정

## 진화 방향 (이 사이클 중에 실제로 발생한 피벗)
- 처음: 로컬 Python/Streamlit + Anthropic Claude
- 중간: Anthropic → **Google Gemini**로 LLM 교체 (비용/접근성)
- 끝: 로컬 앱 → **Vercel 배포 웹앱(Next.js)** 추가. 같은 저장소에 두 버전 공존
- 추가 제약: Vercel free/hobby의 함수 timeout 10초 때문에 회의 길이를 **1분 미만**으로 더 줄임
